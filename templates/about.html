About the Stock Value App

This app is built to satisfy a milestone in TDIs 12 day preparation to transition to Data Science. Through this project I have been exposed to Bokeh, Flask, and Heroku. And while each package started out appearing to be difficult to work with, they turned out to be fairly intuitive and fun to use.

Data for this project comes from the Qandl WIKI dataset, which unfortunately is no longer supported.
